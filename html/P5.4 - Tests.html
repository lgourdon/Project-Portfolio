Shapiro des variables¶

Shapiro-Wilk est un test de normalit.
Il est indiqu pour le traitement d'chantillons rduits
Il retourne les valeurs W = test statistique et p-value = seuil hypothse loi normale

Interprtation

  • Hypothse nulle H0 : la variable suit une loi normale - seuil = 5%
  • Si p-value < : l'hypothse nulle est rejete
  • Si la p-value > : on ne doit pas rejeter l'hypothse nulle

Test de Wilcoxon¶

Le test des rangs signs de Wilcoxon est une alternative non-paramtrique au test de Student pour des chantillons apparis.
Il s'intresse un paramtre de position : la mdiane, le but tant de tester s'il existe un changement sur la mdiane.

Le test de Wilcoxon sur 2 samples est aussi appel test Mann-Whitney

In [14]:
from scipy.stats import mannwhitneyu

# Comparation des samples
stat, p = mannwhitneyu(cl1['edb_2019'], cl5['edb_2019'])
print('Statistics=%.3f, p=%.3f' % (stat, p))

# interprtation
alpha = 0.05
if p > alpha:
	print("Mme distribution (l'hypothse H0 ne peut pas tre rejete)")
else:
	print("Distribution diffrente (rejet de H0)")

Statistics=19.000, p=0.000
Distribution diffrente (rejet de H0)

Hypothse nulle rejete
